# 开源清单（已整理到 `open/`）

## 1. 已包含源码

- `app/src/main/`：Android 主源码与资源
- `app/src/androidTest/`：仪器化测试源码
- `app/build.gradle.kts`
- `app/proguard-rules.pro`
- `build.gradle.kts`
- `settings.gradle.kts`
- `gradle.properties`
- `gradle/libs.versions.toml`
- `.gitignore`
- `.editorconfig`
- `README.md`
- `LICENSE`（Apache License 2.0）

## 2. 建议不要公开

- `local.properties`（本机 SDK 路径）
- `.gradle/`、`.kotlin/`、`build/`、`app/build/`（构建缓存与产物）
- 各类签名文件（`*.jks`、`*.keystore`）与私钥
- 任何账号密钥、API Token、私有服务地址

## 3. 建议在 GitHub 继续补充

- `CONTRIBUTING.md`
- `SECURITY.md`
- `CHANGELOG.md`
- Gradle Wrapper（`gradlew`、`gradlew.bat`、`gradle/wrapper/*`）

- 当前 `open/` 已可作为开源发布的最小源码集。
