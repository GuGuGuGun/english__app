# 任务计划文档：考研离线背单词助手（Android）
## 目标与范围
- 交付一套可离线运行的安卓背单词应用，覆盖学习、查词、词库管理、我的四大模块。
- 支持多词书切换、导入/预置词书、生词本管理与学习进度保持。
- UI遵循 Material Design 3 与极简学术风规范，支持深色模式。
## 约束与原则
- Kotlin + Jetpack Compose + Room + Coroutines/Flow。
- 全离线运行，不申请网络权限，无广告、无外部资源依赖、无发音功能。
- 文件导入/导出使用 SAF，兼容 Android 10+ 分区存储。
- 生词本为“虚拟词书”，可清空但不可删除。
## 里程碑
- M1 项目骨架与数据层打通。
- M2 词书管理与切换、生词本基础逻辑完成。
- M3 学习流程与记忆调度算法完成。
- M4 查词与详情页完成。
- M5 我的页统计、备份/恢复与设置完成。
- M6 UI动效与验收测试完成。
## 详细任务拆解
### 1. 项目初始化
- 建立 Android Studio 工程与基础模块结构。
- 配置 Compose、Room、Coroutines、Material 3 依赖。
- 设置主题、字体与颜色资源。
### 2. 数据层
- Room 实体：Book、Word、Progress、NewWordRef（或等价关联表）。
- DAO：词书、单词、进度、生词关联的基础 CRUD 与查询。
- Database 与预置 vocabulary.db 首次启动拷贝逻辑。
### 3. 词书管理与切换
- 词书列表展示：名称、进度、当前激活状态。
- 一键切换逻辑：激活新词书、保持进度、加载新词。
- 导入 TXT/CSV：文件选择、解析预览、事务写入。
- 删除/清空规则：仅用户导入可删除，生词本仅清空。
### 4. 生词本功能
- getOrCreate 生词本与去重逻辑。
- 学习页与查词页加入入口。
- 学习中“认识”提示移出生词本。
### 5. 核心学习流程
- WordCard 翻转与状态显示。
- 认识/模糊/不认识交互与队列切换。
- SM-2 或等价间隔重复算法工具类。
### 6. 查词模块
- 本地搜索与即时结果列表。
- 详情页与加入生词本操作。
### 7. 我的模块
- 学习统计与日历视图数据结构。
- 数据备份/恢复（导出/导入 DB）。
- 设置项：每组新词数量、字号、深色模式。
### 8. UI与导航
- BottomNav 四 Tab 框架与页面路由。
- 关键组件：WordCard、BookListItem、BottomNavBar。
- 动效：卡片滑动、翻转、星标动效。
### 9. 测试与验收
- 导入解析正确性与异常格式处理。
- 词书切换进度保持、生词本去重验证。
- 离线运行与权限检查。
- 核心流程手动验收：学习、查词、词库管理、我的。
