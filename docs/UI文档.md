# 考研离线背单词助手 - UI设计规范文档 (安卓版)
## 1. 全局设计规范
### 1.1 设计理念
*   **沉浸式**：减少视觉噪点，突出单词本身，适合长时间专注学习。
*   **高对比度**：确保文字清晰，保护视力。
*   **极简操作**：核心功能一步直达，减少层级跳转。
### 1.2 色彩规范
采用护眼双色模式，支持跟随系统深色模式。
**浅色模式**
*   **主色**：`#1A73E8` (学术蓝) - 用于按钮、重点状态。
*   **背景色**：`#F5F5F5` (浅灰) - 页面背景。
*   **卡片色**：`#FFFFFF` (纯白) - 卡片、弹窗背景。
*   **文字主色**：`#212121` (深灰) - 标题、正文。
*   **文字辅色**：`#757575` (中灰) - 描述、音标。
*   **警示色**：`#FF5252` (淡红) - “不认识”按钮、删除操作。
**深色模式**
*   **主色**：`#90CAF9` (柔和蓝)。
*   **背景色**：`#121212` (纯黑/深灰)。
*   **卡片色**：`#1E1E1E` (深灰卡片)。
*   **文字主色**：`#FFFFFF`。
### 1.3 字体规范
*   **单词字体**：`Roboto Medium` 或 `Georgia` (衬线体，显学术感)，建议 `48sp - 56sp` (超大号，视觉中心)。
*   **音标字体**：`Roboto Regular`，`20sp`。
*   **释义字体**：`Noto Sans SC` (思源黑体)，`18sp`。
*   **按钮字体**：`Roboto Medium`，`16sp`。

## 2. 页面详细设计
### 2.1 主框架
**布局**：底部导航栏，包含4个Tab。
*   **Tab 1: 学习** (默认首页) - 图标：`ic_book_open`
*   **Tab 2: 查词** - 图标：`ic_search`
*   **Tab 3: 词库** - 图标：`ic_library`
*   **Tab 4: 我的** - 图标：`ic_person`
---
### 2.2 页面一：学习页
**顶部栏**
*   **左侧**：当前词书名称（如“2024考研大纲”），点击向下箭头可展开**词书切换浮层**。
*   **右侧**：进度指示器（如 `15/100`）。
*   **背景**：统一背景色，无阴影，极简。
**中部卡片区域**
*   **样式**：居中悬浮卡片，圆角 `16dp`，微微阴影。
*   **正面**：
    *   居中大号单词：`abandon`
    *   下方音标：`[əˈbændən]`
    *   提示文案：`点击查看释义`
*   **反面**：
    *   单词与音标（同上，透明度稍降低）。
    *   分割线。
    *   释义文本：`vt. 离弃，丢弃；抛弃`
    *   真题例句：`The study found that people who...` (灰色小字)。
    *   **右上角图标**：星星图标（空心/实心），点击切换“加入生词本”状态。
**底部操作区**
*   **布局**：三个等宽按钮，无背景色，仅保留文字和不同颜色的下划线。
*   **按钮一**：`不认识` (下划线红色) -> 点击后卡片左滑飞出，进度条不增加。
*   **按钮二**：`模糊` (下划线黄色) -> 点击后卡片变淡消失，进度条增加。
*   **按钮三**：`认识` (下划线绿色) -> 点击后卡片右滑飞出，进度条增加。
**词书切换浮层**
*   触发：点击顶部词书名称。
*   样式：底部弹出式面板，圆角顶部。
*   内容：列出所有词书（预置+导入+生词本）。
    *   当前激活词书右侧显示“对勾”图标。
    *   其他词书右侧显示“切换”按钮。
---
### 2.3 页面二：查词页
**搜索栏**
*   样式：通栏搜索框，圆角 `28dp`，带放大镜图标。
*   交互：输入即时显示本地匹配结果。
**搜索结果列表**
*   **Item布局**：
    *   左侧：单词 + 音标。
    *   右侧：简要释义（单行截断） + 星星图标。
*   **交互**：
    *   点击Item：弹出详情页。
    *   点击星星：直接加入生词本（星星变实心，提示“已加入”）。
**单词详情页**
*   样式：全屏或底部滑入面板。
*   内容：完整释义、例句。
*   底部按钮：醒目的“加入生词本”按钮。
---
### 2.4 页面三：词库管理页
**页面标题**
*   标题：“我的词库”，右上角“导入”按钮。
**词书列表**
*   **Item布局**：
    *   **左侧**：词书图标（系统预置显示书本图标，生词本显示星星图标）。
    *   **中间**：
        *   第一行：词书名称（如“考研核心词汇”、“我的生词本”）。
        *   第二行：进度条 + 文字（如 `已学习 200/5000`）。
    *   **右侧**：
        *   若为当前词书：显示“使用中”标签（灰色不可点）。
        *   若非当前词书：显示“切换”按钮（蓝色描边）。
*   **特殊Item：生词本**
    *   置顶显示，背景色微变，不可删除，仅可清空。
**底部悬浮按钮**
*   图标：`+` 号。
*   功能：点击调用系统文件管理器，导入TXT/CSV词书。
**导入解析页**
*   标题：“预览词书”。
*   内容：展示前5行解析数据，供用户确认格式是否正确。
*   底部：两个按钮，“取消”、“确认导入”。
---
### 2.5 页面四：我的
**卡片式布局**
*   **学习统计卡片**：
    *   显示连续打卡天数（大号数字）。
    *   日历视图（标记打卡日期）。
*   **功能列表**：
    *   数据备份：点击导出 `.db` 文件到本地。
    *   数据恢复：点击选择文件恢复。
    *   学习设置：每组新词数量（10/20/30）。
    *   关于软件。
---
## 3. 交互与动效规范
### 3.1 卡片切换动效
*   **认识/模糊**：卡片向右平滑滑出屏幕，同时下一张卡片从左侧淡入滑入。
*   **不认识**：卡片向左平滑滑出，下一张卡片从右侧淡入滑入（暗示需要复习）。
*   **翻转**：点击卡片查看释义时，执行 3D 翻转动画（Duration: 300ms）。
### 3.2 反馈动效
*   **加入生词本**：星星图标点击后变实心，同时伴随一个“放大-缩小”的弹跳效果。
*   **切换词书**：浮层关闭，主页顶部词书名称渐变更新，卡片自动刷新为新书第一个单词。

> "请基于 **Material Design 3** 规范，使用 **Jetpack Compose** 实现上述UI。
>

> 2.  **组件**：
>     *   `WordCard`: 实现翻转动画，正面显示单词，背面显示释义。

>     *   `BottomNavBar`: 底部四个导航项。

> 4.  **图标**：使用 Material Icons，星星图标使用 `Icons.Filled.Star` 和 `Icons.Outlined.Star`。"
